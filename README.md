# Formula1 Statistics

An educational project for visualizing Formula 1 statistics. Built by Dante Raj c:

## Disclaimer

This is an **educational project only** and might not be optimized for the best experience. The data and visualizations provided are for learning purposes.

For the most accurate and up-to-date information, please follow the [official F1 site](https://www.formula1.com).

## Project Setup

This project is built with React and Vite.

### Prerequisites

- Node.js

### Installation

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```

### Development

Run the development server:
```sh
npm run dev
```
